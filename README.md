# Skydrop & Loopback app structure
## Dependencies
1- Nodejs >= 7
2- npm >= 4

## Install

1- Create the Databases in postgresql: skydrop_next, skydrop_next_test (user: next, pass: skydrop_next)
2- Install the dependencies
```
$ cd /path/to/project
$ npm install
```
2- Node . (to run the server)

## Running tests

```
$ npm t
```
