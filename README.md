# Skydrop & Loopback app structure
## Dependencies
* Nodejs >= 7
* npm >= 4

## Install

* Create the Databases in postgresql: skydrop_next, skydrop_next_test (user: next, pass: skydrop_next)
* Install the dependencies
```
$ cd /path/to/project
$ npm install
```
* Node . (to run the server)

## Running tests

```
$ npm t
```
